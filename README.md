# App De Entrada e Saída de Produtos com QR Code

Este é um aplicativo mobile desenvolvido em React Native que oferece uma solução para entrada e saída de produtos usando QR Codes.

## ⚙️ Tecnologias Utilizadas
- ♻️ React Native
